Backend Features:
- Authentication
- CRUD for Spots
- CRUD for Bookings
- CRUD for Reviews
- and more!

Frontend Supports:
- Authentication
- CRUD for Spots
- CRD for Reviews

![](https://github.com/RMPasta/AirBnB-Clone/blob/main/frontend/src/assets/ezgif.com-gif-maker.gif)

Upcoming:

Get Started:
- Copy the ".env-copy" file and enter a db file
- cd into /backend
- npm install
- npm start to run the backend
- cd into /frontend
- npm install
- npm start to run the frontend
